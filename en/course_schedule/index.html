    Course Schedule and Weekly Tasks¶

    Week Tasks Individual responsibilities Submissions
    Week 0 Study lecture notes and theory
    Prepare computer and software environment
    Install required software / set up accounts (MATLAB, Keil Studio Cloud, Processing, Fritzing)    		 All students: read through the entire lecture text; prepare your PC environment
    Student A: check the hardware kit contents (Appendix: parts list)
    Student B: set up Keil Studio Cloud + install Processing (see Week 1; also see Appendix: Processing)
    Student C: install Fritzing + MATLAB (see Week 1)    		 None
    Week 1 Mechanical assembly
    Microcontroller bring-up and debugging
    Breadboard circuit design (Fritzing)
    Integration and debugging    		 Step 1 (individual):
    - Student A: mechanical assembly (see Week 1)
    - Student B: microcontroller bring-up/debug (see Week 1)
    - Student C: Fritzing breadboard design (see Week 1)
    Step 2 (team): integrate and debug successfully (see Week 1)    		 Stand demo video
    Week 2 Build encoder + validate signals
    Close the loop: position control (compare with simulation)    		 Student A: encoder hardware (see Week 2)
    Student B: encoder readout + control on mbed (see Week 2)
    Student C: simulation + comparison (see Week 2)    		 Group report: position-control video; encoder A/B plots; sim vs hardware comparison
    Week 3 Add wheel dynamics and ground friction to simulation
    Add load to the cart/pendulum and test on hardware
    Tune PID gains in MATLAB and try other controllers (LQR/MPC, etc.) and compare performance    		 Student A: attach load and run hardware tests under load, record video (see Week 3)
    Student B: extend the MATLAB/Simscape model (wheel dynamics + ground friction) and document results (see Week 3)
    Student C: tune PID gains under load; try other controllers and compare simulation vs real results (see Week 3)    		 Group report: enhanced simulation results; load test video; PID tuning + controller comparison documentation
    Week 4 Prepare presentation slides
    Demonstrate working system
    Q&A and discussion    		 All students: prepare a 10-minute presentation covering theory, design, simulation, hardware results, challenges; demonstrate live inverted pendulum control; answer technical questions Final submission: slides; final demo video; presentation & Q&A
