# 週ごとの課題と進め方

| **週** | **内容** | **担当の目安** | **提出物** |
|---|---|---|---|
| **Week 0** | テキストと理論背景の確認<br>PC とソフトウェア環境の準備<br>MATLAB / Keil Studio Cloud / Processing / Fritzing の準備 | **全員：** テキスト全体に目を通す／PC 環境を整える<br>**学生A：** ハードウェアキット内容の確認（[付録：部品表](appendix.md#app:partslist)）<br>**学生B：** Keil Studio Cloud の準備と Processing の導入（[第 3.3.2 節](#sec:w1-microcontroller-setup)；[付録：Processing](appendix.md#app:processing)）<br>**学生C：** Fritzing と MATLAB の準備（[第 3.3.3 節](#sec:w1-breadboard-wiring-design)；[第 4.5.2 節](#sec:matlab-setup)） | なし |
| **Week 1** | 機械組立<br>マイコンの立ち上げとデバッグ<br>ブレッドボード配線設計（Fritzing）<br>統合作業とデバッグ | **Step 1（個別）：**<br>- 学生A：機械組立（[第 3.3.1 節](#sec:w1-mechanical-assembly)）<br>- 学生B：マイコン立ち上げとデバッグ（[第 3.3.2 節](#sec:w1-microcontroller-setup)）<br>- 学生C：Fritzing による配線設計（[第 3.3.3 節](#sec:w1-breadboard-wiring-design)）<br>**Step 2（全員）：** 統合とデバッグ（[第 3.4 節](#sec:w1-circuit-assembly)） | 倒立デモ動画 |
| **Week 2** | エンコーダの製作と信号確認<br>位置制御の導入とシミュレーション比較 | **学生A：** エンコーダのハードウェア（[第 4.3 節](#sec:w2-task-a)）<br>**学生B：** mbed 側の読み取りと制御（[第 4.4 節](#sec:w2-task-b)）<br>**学生C：** シミュレーションと比較（[第 4.5 節](#sec:w2-task-c)） | **グループレポート：** 位置制御の動画；エンコーダ A/B 波形；シミュレーションと実機の比較 |
| **Week 3** | 車輪ダイナミクスと床面摩擦の追加<br>負荷付き実機試験<br>MATLAB での PID 調整と他手法比較 | **学生A：** 負荷取り付け／負荷付き実機試験と動画記録<br>**学生B：** MATLAB/Simscape モデル拡張（車輪 + 摩擦）と整理<br>**学生C：** 負荷条件での PID 調整；LQR / MPC 等との比較と実機差の整理 | **グループレポート：** 拡張シミュレーション結果；負荷試験動画；PID 調整と制御比較の記録 |
| **Week 4** | 発表スライド作成<br>実機デモ<br>質疑応答と議論 | **全員：** 理論・設計・シミュレーション・実機結果・課題を含む 10 分発表を準備／倒立制御のライブデモ／技術的な質問に答える | **最終提出：** スライド；最終デモ動画；発表と質疑応答 |

# **※※※ 演習前に読んでください** ※※※

- **本テキストの予習**：演習実施前に，本テキストを読んで，作業の概要や理論的背景を理解しておいてください．テキストを読まずにいきなり作業をすると，演習続行が困難になるような致命的ミスにつながります．

- **補助資料の所在**：今年度版の補助資料（デバッグツール，サンプルプログラム，付録PDF，MATLABモデルなど）は， 以下に整理されています[^1]．特に には，実験で利用する制御プログラムと Processing 用ツールが入っています．

- **パソコンの準備**：今期の実践演習は，設計演習室の PC を利用しますので，自前 PC を用意する必要はありません．一方，自前 PC でも実施可能です．その場合，Windows もしくは Macintosh が必要です．また，制御用マイコンと接続するために USB ポート（Type-A）が必要です．Type-C ポートしかない場合には，Type-A に接続可能なハブを準備してください．

- **作業は当日**：プログラム環境のインストール，機構の組み立て等については，演習当日に説明します．
