# Course Schedule and Weekly Tasks

| **Week** | **Tasks** | **Individual responsibilities** | **Submissions** |
|---|---|---|---|
| **Week 0** | Study lecture notes and theory<br>Prepare computer and software environment<br>Install required software / set up accounts (MATLAB, Keil Studio Cloud, Processing, Fritzing) | **All students:** read through the entire lecture text; prepare your PC environment<br>**Student A:** check the hardware kit contents ([Appendix: parts list](appendix.md#app:partslist))<br>**Student B:** set up Keil Studio Cloud + install Processing (see [Week 1](week1.md); also see [Appendix: Processing](appendix.md#app:processing))<br>**Student C:** install Fritzing + MATLAB (see [Week 1](week1.md)) | None |
| **Week 1** | Mechanical assembly<br>Microcontroller bring-up and debugging<br>Breadboard circuit design (Fritzing)<br>Integration and debugging | **Step 1 (individual):**<br>- Student A: mechanical assembly (see [Week 1](week1.md))<br>- Student B: microcontroller bring-up/debug (see [Week 1](week1.md))<br>- Student C: Fritzing breadboard design (see [Week 1](week1.md))<br>**Step 2 (team):** integrate and debug successfully (see [Week 1](week1.md)) | Stand demo video |
| **Week 2** | Build encoder + validate signals<br>Close the loop: position control (compare with simulation) | **Student A:** encoder hardware (see [Week 2](week2.md))<br>**Student B:** encoder readout + control on mbed (see [Week 2](week2.md))<br>**Student C:** simulation + comparison (see [Week 2](week2.md)) | **Group report:** position-control video; encoder A/B plots; sim vs hardware comparison |
| **Week 3** | Add wheel dynamics and ground friction to simulation<br>Add load to the cart/pendulum and test on hardware<br>Tune PID gains in MATLAB and try other controllers (LQR/MPC, etc.) and compare performance | **Student A:** attach load and run hardware tests under load, record video (see [Week 3](week3.md))<br>**Student B:** extend the MATLAB/Simscape model (wheel dynamics + ground friction) and document results (see [Week 3](week3.md))<br>**Student C:** tune PID gains under load; try other controllers and compare simulation vs real results (see [Week 3](week3.md)) | **Group report:** enhanced simulation results; load test video; PID tuning + controller comparison documentation |
| **Week 4** | Prepare presentation slides<br>Demonstrate working system<br>Q&A and discussion | **All students:** prepare a 10-minute presentation covering theory, design, simulation, hardware results, challenges; demonstrate live inverted pendulum control; answer technical questions | **Final submission:** slides; final demo video; presentation & Q&A |
